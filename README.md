# 🦎 MecchaMapTool: #1 Map Exporter for MECCHA CHAMELEON

MecchaMapTool is an all-in-one utility designed to automate the painful process of creating and exporting custom maps for **MECCHA CHAMELEON**. Stop wasting time creating folders, setting up complex asset collisions, and manually writing configuration files. Build your level, open the tool, and publish to Steam Workshop in seconds!

## 🔥 Core Features
* **1-Click Steam Export:** Automatically detects your MECCHA CHAMELEON installation folder and generates the correct `meta.json` architecture.
* **Smart Texture Baking:** Automatically processes textures and camouflages to perfectly match the game's shader requirements.
* **Blender Automation (Headless Mode):** Integrates directly with Blender in the background to handle mesh collisions and asset optimization without manual tweaking.
* **Zero Dependencies:** Distributed as a standalone compiled file. No Python, Node.js, or complex setups required for end-users.

## 🚀 How to Use (Quick Start)
1. Go to the **Releases** section on the right and download the latest `MecchaMapTool Setup`.
2. Run the tool (it will automatically find your Steam game directory).
3. Select your project folder, enter your map metadata, and click **"Generate & Export"**.
4. Open the official Steam Workshop Tool and upload your fresh map!
